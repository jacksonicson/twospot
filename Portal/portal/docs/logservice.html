LogService

Über den LogService können TwoSpot-Anwendungen Log-Statements ausgeben. Diese werden zentral in der Datenbank zusammengeführt und können über das Portal für jede Anwendung eingesehen werden.

Verwendung

Der Zugriff auf den LogService erfolgt über das Interface LogService. Eine TwoSpot-Anwendung kann über die Factory LogServiceFactory ein konkretes LogService-Objekt laden. Der folgende Codeausschnitt demonstriert dies: 

LogService log = LogServiceFactory.getLogService();

Über das LogService-Objekt lassen sich anschließend Log-Statements mit verschiedenen Prioritäten erstellen. Für jede Priorität existiert eine entsprechende Funktion:

  • void debug(String message);
  • void info(String message);
  • void warn(String message);
  • void error(String message);

Das folgende Code-Beispiel verdeutlicht die Verwendung des LogService

LogService log = LogServiceFactory.getLogService();
logger.error("Log is null");
