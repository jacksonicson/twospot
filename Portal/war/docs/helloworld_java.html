Twospot documentation

TwoSpot HelloWorld (Java)

Setup

Die TwoSpot-Tools sind eine Sammlung von Python Hilfs-Scripten die der vereinfachten Anwendungsentwicklung dienen. Über die Tools lassen sich Anwendungen deployen. Zusätzlich helfen Sie auch beim Anlegen neuer Anwendungen. Die Tools lassen sich hier herunterladen.

Das ZIP-Archiv tools.jar kann ein beliebiges Verzeichnis extrahiert werden. Das Verzeichnis muss in die PATH-Umgebungsvariabel aufgenommen werden, damit der twospot.py-Befehl verwendet werden kann.

Vor dem Deployment einer Anwendung muss für diese eine AppId registriert werden. Die Registrierung erfolgt über das TwoSpot-Portal, das hier erreichbar ist. Nach der Benutzerregistrierung lassen sich drei AppId's registrieren. Nachfolgend wird als AppId der Name gogo verwendet. Dieser muss durch die eigenen AppId ersetzt werden.

Dateistruktur

Die Dateistruktur von TwoSpot-Anwendungen weist sehr große Ähnlichkeit mit der Dateistruktur gewöhnlicher Java Web-Anwendungen auf. Die Dateistruktur der leeren gogo-Anwendungen hat die Form:

 /helloworld
    /WEB-INF
        /web.xml
        /classes
            /class files
            /application configuration files (e.g. the etc directory)
        /lib
            /jar files
    /app.yaml
    /index.html
    /*.jsp
    /...

Die Verzeichnisstruktur lässt sich einfach über die Tools erstellen. Dazu muss der folgende Befehl in dem Verzeichnis ausgeführt werden, in dem die Dateistruktur angelegt werden soll. 

twospot.py --createProject gogo --type java

Jede TwoSpot-Anwendung muss die Datei app.yaml enthalten. Dabei handelt es sich um eine Konfigurationsdatei die Informationen über die Anwendung (AppId, Typ der Runtime) und auch Konfigurationseinstellungen für den Application-Container beinhaltet. Beim Anlegen der Verzeichnisstruktur über die Tools wird die app.yaml-Datei automatisch erstellt. Für die HelloWorld-Anwendung sind keine Änderungen an der Datei notwendig.

Startseite

Im Anwendungsverzeichnis kann nun eine einfache HTML-Seite als Startseite für die TwoSpot-Anwendung angelegt werden. Für die gogo-Anwendung wurde die folgende index.jsp-Datei angelegt:

<html>
   <body>
      <h1>GoGo - HelloWorld</h1>
   </body>
</html>

Deployment

Nach dem Anlegen der Startseite kann die TwoSpot-Anwendung in die Platform deployt werden. Das Deployment erfolgt ebenfalls über die Tools. Der folgende Befehl muss im Verzeichnis der gogo-Anwendung ausgeführt werden. Beim Deployment fragt das Script nach dem Benutzernamen und Passwort. 

twospot.py --deploy
