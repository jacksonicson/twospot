HelloWorld in Java

Requirements

Download the Tools package and extract it to a folder of your choice. The destination folder has to bee in your PATH variable. You can check this by executing twospot.py -h in the command line. If the command is found and you see the twospot.py help information, everythin is ok.

The folder structure for the HelloWorld application is created by using the twospot.py tools script. The command twospot.py --createApplication HelloWorld --type java crates the folder structure for a java application named HelloWorld.

The genrated folder structure looks very similar to a standard java web application. The only difference lies in the app.yaml configuration file. This file contains platform specific configuration settings.
